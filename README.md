## Troubleshooting

| Issue | Cause | Fix |
|---|---|---|
| Auth fails | Invalid API key | Re-export key |
| Timeout | Network latency | Increase timeout |
| Empty output | Bad prompt | Check template |
| Rate limit | Too many requests | Add delay |
